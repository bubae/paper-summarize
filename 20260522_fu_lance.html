3.1 м„Өкі„ мӣҗм№ҷ: unified context learning + decoupled capability pathways

Unified Context Learning: heterogeneous мһ…л Ҙмқ„ shared interleaved multimodal sequenceлЎң ліҖнҷҳ.

  • Text: Qwen2.5-VLмқҳ language embedding layerлЎң embed.
  • Understanding-oriented visual: ##Qwen2.5-VL ViT encoder##(14Г— spatial / 2Г— temporal patching + 2Г—2 spatial merge)лЎң compact semantic visual token. language-aligned semantics м ңкіө.
  • Generation-oriented visual: ##Wan2.2 3D causal VAE encoder##лЎң continuous latent(16Г— spatial downsampling, 4Г— temporal downsampling for video). low-level appearanceВ·temporal structure ліҙмЎҙ, lightweight MLP connectorлЎң generation backbone hidden spaceм—җ projection.

к°Ғ sampleмқҖ unified interleaved sequenceлЎң н‘ңнҳ„ (мӢқ 1вҖ“2):

$$\mathcal{S} = \cdots \oplus \mathcal{B}_{\text{text}}(\mathbf{T}) \oplus \mathcal{B}_{\text{vis}}(\mathbf{V}_{\text{vit}}) \oplus \mathcal{B}_{\text{vis}}(\mathbf{V}_{\text{vae}}^{\text{clean}}) \oplus \mathcal{B}_{\text{vis}}(\mathbf{V}_{\text{vae}}^{\text{noisy}}) \oplus \mathcal{B}_{\text{text}}(\mathbf{T}') \oplus \cdots$$

$$\mathcal{B}_{\text{text}}(\mathbf{T}) = [\texttt{BOT}, \mathbf{T}, \texttt{EOT}], \quad \mathcal{B}_{\text{vis}}(\mathbf{V}) = [\texttt{BOV}, \mathbf{V}, \texttt{EOV}]$$

м§ҒкҙҖм ҒмңјлЎң л§җн•ҳл©ҙ, н…ҚмҠӨнҠёВ·мқҳлҜё мӢңк°ҒнҶ нҒ°В·мғқм„ұмҡ© мӢңк°ҒнҶ нҒ°мқ„ лӘЁл‘җ н•ҳлӮҳмқҳ кёҙ sequenceм—җ м„һм–ҙ лҶ“лҗҳ, к°Ғ modalityлҘј special token(BOT/EOT, BOV/EOV)мңјлЎң кІҪкі„ м§“лҠ”лӢӨ. generalized 3D causal attention: sequenceлҘј modality-specific segmentлЎң л¶„н• , к°Ғ segmentлҠ” м„ н–ү segmentм—җ attend(causal dependency ліҙмЎҙ). segment лӮҙл¶Җм—җм„ң ##text tokenмқҖ causal attention, visual tokenмқҖ bidirectional attention##(spatialВ·spatiotemporal structure нҸ¬м°©).

Decoupled Capability Pathways: лӘЁл“  modalityлҘј shared sequenceм—җ мЎ°м§Ғн•ҳлҗҳ, understandingВ·generationмқ„ specialized expert pathwayлЎң мІҳлҰ¬.

  • Generation expert $\text{LLM}_{\text{GEN}}$: VAE latent tokenм—җ мһ‘лҸҷ, interleaved context мЎ°кұҙмңјлЎң generation-side hidden state мҳҲмёЎ. clean latent $x_1$, noise $x_0 \sim \mathcal{N}(0,I)$, interpolated latent $x_t = t x_1 + (1-t) x_0$ ($t \sim \mathcal{U}(0,1)$). flow matching objective (мӢқ 4): $$\mathcal{L}_{\text{GEN}} = \mathbb{E}_{x_0, x_1, t}\left[\|v_{\theta_{\text{GEN}}}(x_t, \mathcal{S}, t) - (x_1 - x_0)\|_2^2\right]$$

3.3 Modality-Aware Rotary Positional Encoding (MaPE)

unified multimodal trainingм—җм„ңлҠ” single sequenceк°Җ ##м—¬лҹ¬ visual token group($\mathbf{V}_{\text{vae}}^{\text{noisy}}, \mathbf{V}_{\text{vit}}, \mathbf{V}_{\text{vae}}^{\text{clean}}$)##мқ„ нҸ¬н•Ён•ңлӢӨ. мқҙл“ӨмқҖ source encoderлҝҗ м•„лӢҲлқј functional roleлҸ„ лӢӨлҘҙлӢӨ(semantic cue / visual condition / generation target). н‘ңмӨҖ 3D-RoPEлҠ” spatiotemporal layoutл§Ң encodeн•ҳлҜҖлЎң мқҙл“Өмқ„ кө¬л¶„ лӘ» н•ҙ ##positional ambiguity##мҷҖ cross-task alignment м•Ҫнҷ” мң л°ң.

Qwen2.5-VLмқҳ мӣҗлһҳ 3D-RoPEм—җм„ң мӢңк°Ғ нҶ нҒ° $(t,h,w)$мқҳ мң„м№ҳ (мӢқ 6): $\hat{\mathbf{p}}^{\text{vis}}_{t,h,w} = D + [t, h, w] = [D+t, D+h, D+w]$, $t \in [0,T-1], h \in [0,H-1], w \in [0,W-1]$.

MaPEлҠ” ##к°Ғ modality group $m_i$мқҳ index $i$м—җ л”°лқј temporal м°Ёмӣҗм—җл§Ң fixed offset $\Delta_t$лҘј м Ғмҡ©## (мӢқ 7):

$$\mathbf{p}^{(m_i)}_{t,h,w} = \hat{\mathbf{p}}^{(m_i)}_{t,h,w} + [i \cdot \Delta_t, 0, 0] = [\hat{t}^{(m_i)}_{t,h,w} + i\cdot\Delta_t,\ \hat{h}^{(m_i)}_{t,h,w},\ \hat{w}^{(m_i)}_{t,h,w}],\quad i \in \{0,1,2\}$$

м§ҒкҙҖ: м„ё мў…лҘҳмқҳ мӢңк°Ғ нҶ нҒ° к·ёлЈ№мқ„ ##мӢңк°„м¶•м—җм„ң м„ңлЎң лӢӨлҘё м¶ңл°ңм җ($i\cdot\Delta_t$)м—җ л°°м№ҳ##н•ҳм—¬ "мқҙкұҙ мқҳлҜёмҡ©, мқҙкұҙ мЎ°кұҙмҡ©, мқҙкұҙ мғқм„ұ нғҖк№ғ"мһ„мқ„ мң„м№ҳ м •ліҙлЎң лӘ…мӢңн•ңлӢӨ. spatial мўҢн‘ңлҠ” к·ёлҢҖлЎң л‘җм–ҙ image/videoмқҳ intrinsic layoutкіј temporal orderВ·мғҒлҢҖ кұ°лҰ¬лҘј ліҙмЎҙ. $\Delta_t = 1000$мңјлЎң м„Өм •.

мҷң VE/VAE м ңкұ°к°Җ м–ҙл Өмҡҙк°Җ

VE (CLIP, SigLIP, DINOv2)лҠ” ##discriminative pretraining objective##м—җм„ң н•ҷмҠөлҗҳм–ҙ ##high-level semantic feature##л§Ң retainн•ҳкі  fine-grained pixel detailмқҖ lossy. л°©лІ• (Method): 3-Layer Architecture

Layer 1: Near-Lossless Visual Interface

Patch Encoding Layer

мһ…л Ҙ image (лҳҗлҠ” noise)лҘј ##2-layer convolution (GELU activation, 2D sinusoidal positional encoding)##лЎң visual tokenмңјлЎң л§Өн•‘. Conv stride 16, 16 вҶ’ 32Г—32 image patch н•ҳлӮҳк°Җ н•ҳлӮҳмқҳ visual tokenмңјлЎң. ##<img>/</img> special token##мңјлЎң visual content delimit.

мӨ‘мҡ”: ##downsampling ratio 32##мқёлҚ°лҸ„ unified modelмқҙ strong understanding+generationмқ„ мң м§Җ. мқҙлҠ” token-level pixel-word correspondenceлҘј deeply correlated learningмңјлЎң ліҙмғҒ.

Patch Decoding Layer

  • Understanding stream: linear projection head вҶ’ token-to-vocabulary.
  • Generation stream: ##multi-layer MLP headлЎң pixel patch м§Ғм ‘ мҳҲмёЎ##. ##diffusion headлӮҳ VAE decoderлҘј мҡ°нҡҢ##.

Dynamic Noise Scale

Generation streamмқҙ ##varying resolution##м—җм„ң мһ‘лҸҷн•ҳлҜҖлЎң naive unit-variance prior $z_1 \sim \mathcal{N}(0, I)$к°Җ signal scaleкіј mismatch. Resolution-adaptive noise scale:

$$\sigma_R(H,W) = \sigma_0 \sqrt{N(H,W) / N_0} \quad (\text{NSEmb})$$

$N(H,W) = (H \cdot W)/32^2$лҠ” generation token мҲҳ, $N_0$лҠ” reference count. м§ҒкҙҖм ҒмңјлЎң л§җн•ҳл©ҙ, ##кі н•ҙмғҒлҸ„ imageлҠ” лҚ” л§ҺмқҖ noiseлҘј л°ӣм•„м•ј## per-token noise energyк°Җ мқјм •н•ҳкІҢ мң м§Җ. Conditioning signal:

$$s_t = \tau_t + \text{NSEmb}(\tilde\sigma(H,W)), \quad \tilde\sigma = \sigma_R/\sigma_\max \in [0,1]$$

Layer 2: Native Mixture-of-Transformers (MoT)

Improved Native Primitive

Native RoPE multi-axis: text tokenмқҖ ##temporal axis T##л§Ң мӮ¬мҡ© (H=W=0), image tokenмқҖ ##м¶”к°ҖлЎң spatial axes H, W##. Q/K projectionмқҳ head dimensionмқ„ T, H, WлЎң ##reallocate##, к°Ғк°Ғ independent frequency base. н…ҚмҠӨнҠёмҷҖ л№„мЈјм–ј modelingмқ„ лӢЁмқј unified attention м•Ҳм—җм„ң nativeн•ҳкІҢ м§Җмӣҗ.

Two Streams, One Sequence

Understanding streamкіј generation streamмқҖ ##лӘЁл“  layerм—җм„ң shared self-attention##:

  • Clean imageмҷҖ textлҠ” understanding streamмңјлЎң мІҳлҰ¬ (causal w.r.t. text, bidirectional within image block).
  • Noise-conditioned inputмқҖ generation streamмңјлЎң (noise tokensк°Җ same image block м•Ҳм—җм„ң bidirectional, full access to clean inputs).
  • Clean tokensлҠ” noise tokensлЎңл¶Җн„° attendн•ҳм§Җ лӘ»н•Ё (prevent leak).

##Full parameter decoupling##: л‘җ streamмқҖ ##separate projections, normalizations, feedforward block##мқ„ к°Җм§Җл©° ##к°Ғ layerм—җм„ң token typeм—җ л”°лқј dynamic routing##.

Layer 3: Joint Training Objective

Total Loss (Equation 1)

$$\mathcal{L}_\text{total} = \lambda_1 \mathcal{L}_\text{Und} + \lambda_2 \mathcal{L}_\text{Gen}$$

Autoregressive Text Loss (Equation 2)

$$\mathcal{L}_\text{Und} = -\frac{1}{N}\sum_{n=1}^{N} \log p_\theta(x_n | x_{

н‘ңмӨҖ next-token prediction.

Pixel-Space Flow Matching (Equations 3вҖ“5)

JiT-style x-predict + v-loss. Clean image $\mathbf{x}$мҷҖ Gaussian sample $\boldsymbol\epsilon \sim \mathcal{N}(0, I)$:

$$\mathbf{z}_t = t \mathbf{x} + (1-t) \sigma_R \boldsymbol\epsilon, \quad t \in [0,1] \quad (3)$$

$t=0$мқҙ pure noise, $t=1$мқҙ clean image. Velocity term:

$$\mathbf{v}_\theta(\mathbf{z}_t, t) = \frac{\hat{\mathbf{x}}_\theta(\mathbf{z}_t, t, \mathbf{s}_t) - \mathbf{z}_t}{1-t} \quad (4)$$

MSE нҳ•нғң loss:

$$\mathcal{L}_\text{Gen} = \mathbb{E}_{t, \mathbf{x}, \boldsymbol\epsilon, (H,W)}\left[ \|\mathbf{v}_\theta(\mathbf{z}_t, t) - \mathbf{v}^\star\|_2^2 \right], \quad \mathbf{v}^\star = \frac{\mathbf{x} - \mathbf{z}_t}{1-t} \quad (5)$$

м§ҒкҙҖм ҒмңјлЎң л§җн•ҳл©ҙ, modelмқҙ ##noise вҶ’ clean trajectoryмқҳ velocity##лҘј н•ҷмҠө.

Unified Classifier-Free Guidance (Equation 6)

Text condition $c_\text{txt}$мҷҖ image context $c_\text{img}$лҘј ##independently modulate##:

$$\nabla_\mathbf{x} \log p(\mathbf{x} | c_\text{img}, c_\text{txt}) = \gamma(\nabla_\mathbf{x} \log p(\mathbf{x}|c_\text{img}, c_\text{txt}) - \nabla_\mathbf{x} \log p(\mathbf{x}|c_\text{img})) + \gamma_\text{img}(\nabla_\mathbf{x} \log p(\mathbf{x}|c_\text{img}) - \nabla_\mathbf{x} \log p(\mathbf{x})) + \nabla_\mathbf{x} \log p(\mathbf{x})$$

Training мӨ‘ textлҘј 10% drop, text+imageлҘј м¶”к°Җ 10% drop. Empirically $\gamma=4, \gamma_\text{img}=1$мқҙ best вҖ” ##modelмқҙ visual conditioningмқҖ effectively capture, text alignmentм—җ лҚ” к°•н•ң guidance н•„мҡ”##.

Stage 1: Understanding Warmup

Pretrained NEOм—җм„ң мҙҲкё°нҷ”. 2-phase:

  • Attention-Fusion Phase: NEOмқҳ QK projectionкіј normalizationмқ„ ##temporal axisмҷҖ spatial axesм—җ кұёміҗ single shared setмңјлЎң unify## (QK parameter footprintлҘј м Ҳл°ҳмңјлЎң). RoPE frequency: temporal $\theta=5{,}000{,}000$, spatial $\theta=10{,}000$. ##кё°мЎҙ networkлҠ” freeze, attention layerл§Ң fine-tune##.
  • Full-Model Continuation Phase: м „мІҙ understanding branch unfreeze, mid-training corpusлЎң LR 2e-5. SenseNova-U1мқҳ ##understanding backbone## кө¬м¶•.

Stage 2: Generation Pre-Training (3 Phases)

  • Phase I: text-to-image data 256ВІ~1024ВІ resolution, constant LR 2e-4, 120K steps.
  • Phase II: resolution вүҘ 512ВІ, 60K steps, LR 1e-4.
  • Phase III: editing + reasoning + interleaved image-text generation м¶”к°Җ, 120K steps, cosine LR 1e-4 вҶ’ 2e-5.

A3B variantлҠ” MoE balance loss coefficient $5 \times 10^{-3}$.

Stage 3 & 4: Unified Joint Training

Both branchesк°Җ ##end-to-end jointly optimize##. Data sampling ratio: understanding 0.33, generation 0.37, editing 0.24, interleave 0.06. Loss weights $\lambda_1=0.1, \lambda_2=1.0$ (generationмқҙ dominate). Stage 4 SFTлҠ” ##9K steps##лЎң лҚ” м§§кі  carefully curated dataлЎң instruction alignment.

Stage 5: Post-Training for T2I

##Flow-GRPO##лҘј мӮ¬мҡ©н•ң RL: text rendering reward (PaddleOCR IoU), style following reward (VLM judge), aesthetic reward (HPSv3). л‘җ reward group көҗлҢҖ н•ҷмҠө:

  • Group 1: text rendering + style ($r = r_\text{ocr} + \lambda_\text{sty} r_\text{sty}$, $\lambda_\text{sty}=0.5$).
  • Group 2: aesthetic only.

Dynamic resolution warmup: difficulty score кё°л°ҳ gated sampling (Equation 7). 8B variant 1,600 epochs, A3B variant 200 epochs.

Stage 6: CFG & Step Distillation (DMD2)

Distribution Matching Distillation v2лЎң NFE ##100 вҶ’ 8##. Three model: generator $G$, fake flow model $F$, teacher $T$ (лӘЁл‘җ teacherм—җм„ң initialize). $G$лҠ” ##5 update of $F$л§ҲлӢӨ 1лІҲ update##. AdamW lr 2e-6 (G), 4e-7 (F).

Inference Infrastructure: Disaggregated Deployment

л‘җ specialized open-source engine:

  • LightLLM: multimodal understanding, text streaming, request orchestration. LLM-oriented tensor parallelism (TP).
  • LightX2V: image generation. Diffusion-oriented (CFG parallelism, sequence parallelism).

л‘җ engineмқҙ ##pinned shared memory##лЎң generation state көҗнҷҳ. Three practical benefits: (1) different parallelism per workload, (2) independent resource budget, (3) operational isolation.

Hybrid Attention Kernel

Unified multimodal prefillмқҳ system challenge: text rowsлҠ” causal, image rowsлҠ” ##full text prefix + full image span##м—җ attend. ##Optional b_image_token_end##лЎң M-block лӢЁмң„ masking decision: image tokenмқҙ мһҲмңјл©ҙ key rangeлҘј image-span endк№Ңм§Җ нҷ•мһҘ, м—Ҷмңјл©ҙ causal fast path. Triton kernel + modified FlashAttention3 л‘җ backend.

TL;DR

  • 문제: ##Tool-Integrated Reasoning (TIR)##을 small language model (SLM, ≤2B)에 scale하는 것은 어렵다 -- (a) GRPO 등 RL은 ##trajectory-level sparse reward##만 제공하며 약한 exploration 능력의 SLM은 cold-start regime에 갇힘; (b) On-policy distillation (OPD)는 dense token-level supervision을 주지만 ##TIR에 적용하면 critical failure mode##: ##erroneous tool call이 cascade##되어 student가 OOD state로 빠지고, 그 영역에서 teacher의 token-level supervision이 ##unreliable##해진다 (entropy 폭증). 이는 ##discontinuous tool observation##이 text-only reasoning과 다른 distribution shift를 만들기 때문.
  • 해결: ##SOD (Step-wise On-policy Distillation)## -- step-level student-teacher divergence를 직접 측정하여 ##각 step의 distillation strength를 adaptively reweight##. Student가 teacher와 well-aligned이면 dense supervision 그대로, drift가 커지면 ##misleading high-divergence region에서 distillation signal을 progressively attenuate##. ##GRPO outcome reward + step-wise reweighted OPD##를 함께 사용 ($\mathcal{L} = \mathcal{L}_{\text{GRPO}} + \mathcal{L}_{\text{OPD}}^{\text{step}}$).
  • 왜 작동하는가: 두 propositional 결과로 진단. (Prop 1) ##Discontinuous divergence amplification##: text-only는 $\Delta_{k+1} - \Delta_k = O(\eta)$, TIR은 single erroneous tool observation으로 $\Omega(m \cdot \eta_{\text{tool}})$, $j$ consecutive errors → super-linear compound. (Prop 2) ##Gradient SNR degradation##: teacher-supported overlap $\rho_t \leq \rho$이면 OPD loss의 second moment $\geq (1-\rho) \log^2(1/\epsilon)$, SNR → 0. SOD의 reweighting이 weighted second moment를 $O((d_1/d_k)^2)$로 suppress해서 SNR collapse 복구.
  • 비유: 운전 강사(teacher)가 학생(student) 차에 동승해 매 순간 교정 지도. 학생이 길에서 안 벗어나면(low divergence) 모든 지시가 유효하지만, 학생이 ##낭떠러지로 차를 모는 도중##(high divergence)에는 ##강사의 일반 도로 코칭이 위험##해진다 -- "가속 페달 더 밟아!"가 절벽에서는 치명적. SOD는 ##일탈이 클 때 코칭 강도를 줄이고, 학생이 다시 도로로 돌아오면 강도를 회복##한다.

핵심 기여 (Key Contributions)

  1. SLM TIR에서 OPD failure mode 진단: ##erroneous tool call cascade → student-teacher divergence super-linear amplification + teacher entropy 폭증##을 800 trajectory에서 정량 입증 (Figure 1a, 1b). Text-only reasoning과의 ##정성적 차이##를 명료화.
  2. SOD 알고리즘: Step-level divergence proxy $d_k$ + ##multiplicative ratio-based reweighting## $w_k = \min(\prod_{u=1}^{k-1} (d_u + \epsilon)/(d_{u+1} + \epsilon), 1+\delta)$ (Eq.7). divergence가 ##꺽이면(recovery)## weight가 다시 amplify되어 ##temporary error에 over-penalize 안 함##.
  3. 이론적 정당화: Prop 1 (discontinuous divergence amplification, super-linear compound), Prop 2 (gradient SNR degradation), SOD의 weighted second moment $O((d_1/d_k)^2)$ suppression -- monotonically increasing divergence 가정 하에 ##bounded gradient SNR 복구##.
  4. SOTA on SLM TIR: ##0.6B student가 AIME 2025에서 26.13% (avg@32)## 달성 -- 저자 주장상 ##sub-billion-parameter 모델이 이 수준 도달은 처음##. 1.7B는 50.83/41.72/38.72/40.63 (AIME24/25/GPQA-D/LiveCodeBench), 평균 ##42.98##로 second-best baseline (OPD 36.27) 대비 ##+18.50% 상대 개선##. 0.6B는 24.22 vs OPD 20.04 (##+20.86%##).
  5. Ablation으로 component 분리: Uniform / heuristic exponential decay / hard mask after wrong / no clipping 4가지 reweighting variant 모두 SOD에 미달 (34.70/37.14/31.85/38.10 vs SOD 42.98). GRPO 또는 step-wise OPD 단독 제거시 40.78/25.39 -- ##두 항 모두 indispensable##.
  6. Three Distillation Patterns: SOD가 자연 발현하는 3 패턴 -- ##(1) Stable## (high $d_k$ unchanged → dense supervision 활용), ##(2) Erroneous## (분기 시 weight progressively reduce → misleading suppression), ##(3) Recovery## (drift 후 복귀 시 weight 다시 회복 → "recovery from earlier errors").
  7. Stronger teacher 활용 능력: OPD는 4B → 14B teacher 변경 시 0.6B student 성능 ##저하## (capacity gap amplifies misalignment), SOD는 ##일관 향상## -- richer knowledge harness 가능.

1. Introduction

Agentic capability는 LLM 응용 영역을 크게 확장했지만, 보통 large-scale 모델 의존 → ##비싼 inference, 큰 deployment overhead##. Latency/resource/privacy-sensitive 시나리오에서는 ##on-device SLM##이 실용적으로 중요하나, ##stable + effective Tool-Integrated Reasoning (TIR)## 을 SLM에 transfer하는 것은 미해결 과제.

기존 post-training 접근:

  • RL (GRPO 등): SLM-based TIR에서 unstable. TIR은 long-horizon trajectory + multi-step decision making + tool interaction → ##sparse outcome-level reward## + 작은 모델의 약한 exploration → ##cold-start regime##.
  • OPD: dense token-level supervision으로 sparse-reward 문제 완화하지만, ##TIR에 직접 transfer 시 severe training instability##.

저자들의 핵심 관찰 (Figure 1a, 1b):

  • Text-only reasoning에서 student가 teacher에서 점진적으로 drift해도 ##subsequent state가 generated context를 따라 continuously evolve## → distribution shift가 progressive
  • TIR에서 ##single erroneous tool call이 incorrect observation을 context에 inject## → subsequent reasoning이 ##corrupted state에서 unfold##. SLM은 limited capacity와 weak exploration으로 multiple erroneous tool call accumulate → state distribution이 teacher distribution에서 ##rapidly drift##
  • 이런 OOD state에서 ##teacher token-level supervision은 unreliable / misleading##; 확대된 student-teacher discrepancy → unstable gradients + training collapse

SOD의 핵심 아이디어: "각 reasoning step에서 student-teacher divergence에 따라 distillation strength를 adaptively adjust". 정렬되어 있을 때는 dense supervision 보존, 큰 deviation에서는 distillation signal을 ##progressively attenuate## → high-mismatch region에서 misleading supervision 영향 감소 + well-aligned region에서 informative guidance 유지.

실험 결과: AIME 2024/2025, GPQA-Diamond, LiveCodeBench에서 baseline 대비 ##최대 +20.86%##. 0.6B-scale student가 ##AIME 2025에서 26.13%## -- "to our best knowledge, ##sub-billion-parameter 모델이 이 challenge benchmark에서 이 수준 도달은 처음##".

3. Preliminaries

3.1 Multi-turn Tool-Integrated Reasoning (TIR)

SLM이 외부 환경과 multi-step interaction. Step $k$에서 모델이 response $y_k$ 생성 (natural-language reasoning, tool invocation, 또는 final answer). Tool 호출 시 environment가 observation $o_k$ 반환, context에 append되어 후속 generation 조건. Trajectory:

3.2 GRPO (Group Relative Policy Optimization)

Outcome-level reward $r(\tau)$로 group-relative advantage 계산. Group $\{\tau_i\}_{i=1}^G$, reward $r_i = r(\tau_i)$, advantage:

$$\hat{A}_i = \frac{r_i - \text{mean}(\{r_j\})}{\text{std}(\{r_j\}) + \epsilon_A} \tag{2}$$

Token-level importance ratio $\rho_{i,t}(\theta) = \pi_\theta(y_{i,t}|y_{i, $$\mathcal{L}_{\text{GRPO}} = -\mathbb{E}\left[\frac{1}{G}\sum_{i=1}^G \frac{1}{|\mathcal{T}_i|}\sum_{t \in \mathcal{T}_i} \min(\rho_{i,t}(\theta) \hat{A}_i, \text{clip}(\rho_{i,t}(\theta), 1-\epsilon, 1+\epsilon)\hat{A}_i)\right] \tag{3}$$

$\mathcal{T}_i$ = model-generated token positions (tool observation 제외). ##Sparse outcome-level reward##만 제공.

3.3 On-policy Distillation (OPD)

Student-generated trajectory에서 dense token-level supervision (student visited state에서 reverse KL):

$$\mathcal{L}_{\text{OPD}} = \mathbb{E}\left[\sum_{t \in \mathcal{T}_i} (\log \pi_\theta(y_t | y_{이는 ##student-visited state에서 student/teacher policy의 reverse KL divergence##의 sampled estimator. Sparse reward 문제 완화 + sample efficiency 향상.

4. Methodology

4.1 OPD의 Tool-Induced State Drift Failure

OPD objective는 student-visited state 전반에서 teacher supervision이 reliable이라는 가정. Text-only에서는 distribution drift가 gradual하므로 성립, ##TIR에서는 tool observation이 도입한 discontinuous state transition으로 severely violated##.

Step-level mismatch 정의:

$$\Delta_k = \frac{1}{|\mathcal{I}_k|}\sum_{t \in \mathcal{I}_k} D_{\text{KL}}(\pi_\theta(\cdot|y_{두 propositional 결과 (Appendix D):

Proposition 1 (Discontinuous divergence amplification): Text-only reasoning에서 $\Delta_{k+1} - \Delta_k = O(\eta)$ ($\eta$ = per-token distributional shift). TIR에서 length $m$의 single erroneous tool observation은 $\Delta_{k+1} - \Delta_k = \Omega(m \cdot \eta_{\text{tool}})$, $\eta_{\text{tool}} \gg \eta$. ##$j$ consecutive tool errors 하에 super-linear compound##: $\Delta_{k+j} - \Delta_k = \Omega(\sum_{i=0}^{j-1} m_i \cdot \eta_{\text{tool}}^{(i)})$ with $\eta_{\text{tool}}^{(i+1)} \geq \eta_{\text{tool}}^{(i)}$.
Proposition 2 (Gradient SNR degradation): Teacher-supported region $S_t^\epsilon = \{v \in \mathcal{V}: \pi_{\text{teacher}}(v|y_{

이 두 결과가 TIR-specific failure cascade 특성화: consecutive erroneous tool call이 compounding divergence amplification (Prop 1) 야기 → low-overlap state로 student를 progressively push → OPD gradient가 high-variance, uninformative contribution에 dominated (Prop 2). Figure 1에서 empirical confirm: error accumulate 시 divergence sharp acceleration (a), teacher entropy elevated and unstable (b).

OPD가 step 전반에 ##loss를 uniformly aggregate##하므로 corrupted signal을 systematically overweight → ##step-wise reweighting의 motivation##. Appendix D.4에서 monotonically increasing divergence 하에 reweighting이 weighted second moment를 $O((d_1/d_k)^2)$만큼 suppress → low-overlap state에서도 ##bounded gradient SNR 복구##.

4.2 Student-teacher Divergence and Step-wise Reweighting

Generated trajectory를 $K+1$ reasoning step으로 partition (각 step = two tool observation 사이의 model response, 또는 final answer step). $\mathcal{I}_k$ = $k$-th model-generated step의 token position set. Tool observation token은 excluded (environment가 produce).

Step-level divergence score:

$$d_k = \frac{1}{|\mathcal{I}_k|}\sum_{t \in \mathcal{I}_k} |\log \pi_\theta(y_t|y_{$d_k$는 teacher supervision의 ##local reliability indicator##. 작으면 student가 teacher distribution과 well-aligned, 크면 substantial mismatch (tool-induced drift나 corrupted observation).

Adaptive reweighting (핵심 수식): 첫 step weight $w_1 = 1$. $k \geq 2$의 weight:

$$w_k = \min\left(\prod_{u=1}^{k-1} \frac{d_u + \epsilon}{d_{u+1} + \epsilon}, 1+\delta\right) \tag{7}$$

$\epsilon$ = stability constant, $\delta$ = maximum amplification 제어. ##Eq.7은 consecutive divergence score의 ratio에만 의존##하므로 absolute value가 아닌 ##$\Delta_k$의 monotone proxy##면 충분 -- $d_k$가 $\Delta_k$와 monotonically consistent하고 OPD forward pass에서 zero marginal cost로 계산되어 implementation에 활용 (Appendix D.5).

직관: $d_{u+1} > d_u$ (학생이 teacher에서 drift away)이면 ratio < 1 → weight 감소 → high-mismatch region에서 distillation 약화. ##Recovery##: $d_{u+1} < d_u$ (학생이 teacher-supported region으로 partially move back)이면 ratio > 1 → weight 다시 증가 → ##corrected step의 supervision 보존##. 이를 ##"recovery from earlier errors"## 현상이라 부른다. $1+\delta$ upper bound가 stable optimization 보장.

$$\mathcal{L}_{\text{OPD}}^{\text{step}} = \mathbb{E}_{y \sim \pi_\theta}\left[\sum_{k=1}^{K+1} w_k \sum_{t \in \mathcal{I}_k}(\log \pi_\theta(y_t|y_{최종 training objective:

$$\mathcal{L} = \mathcal{L}_{\text{GRPO}} + \mathcal{L}_{\text{OPD}}^{\text{step}} \tag{10}$$

$\mathcal{L}_{\text{GRPO}}$가 sparse outcome-level reward로 ##trajectory exploration##, $\mathcal{L}_{\text{OPD}}^{\text{step}}$가 ##student-teacher divergence로 reweighted된 dense token-level guidance## 제공. tool-induced state drift 하에서 stable learning.

5.6 Three Distillation Patterns (Figure 5)

Training 중 SOD가 자연 발현하는 3 패턴:

  1. Stable Pattern (Normal Distillation): Student가 teacher와 closely aligned, $d_k$ low → adaptive weight high → ##teacher supervision의 full utilization##. "sufficient distillation"과 유사.
  2. Erroneous Pattern (Error Suppression): Persistent deviation으로 step-level divergence 증가 (incorrect intermediate reasoning이나 tool usage 원인) → adaptive weight progressively reduce → ##misleading supervision suppress##, corrupted state 하에서 stabilize.
  3. Recovery Pattern (Error Recovery): Student가 처음에 deviate but recover. Weight가 high-divergence steps 동안 attenuate, 이후 restore → ##corrected step의 effective guidance + recovery 후 useful supervision 보존##. Temporary error에 over-penalize 회피.

6. Limitations

  • ##Python code interpreter가 있는 agentic TIR task에 focus## -- web browsing, API call 등 다른 agentic setting의 distribution shift는 unexamined
  • ##Qwen3 family만 사용## -- 강한 안정 성능, multi-scale availability, OPD literature wide adoption으로 fair comparison 위해 선택. 다른 model family validation은 future work.

7. 비판적 분석

강점

  • Concrete failure mode 진단: 800 trajectory에서 ##Figure 1a/b의 정량적 evidence##로 OPD-on-TIR의 실패가 단순 anecdote가 아님을 입증. text-only vs TIR의 정성적 차이를 ##discontinuous tool observation##으로 명료히 설명.
  • 이론적 정당화의 깔끔함: Prop 1 (super-linear divergence amplification)과 Prop 2 (gradient SNR collapse)이 실패와 SOD 해결을 ##단일 framework##에서 설명. Appendix D.4의 weighted second moment $O((d_1/d_k)^2)$ suppression bound는 reweighting 설계의 근거.
  • Multiplicative ratio reweighting의 우아함: Eq.7이 ##absolute divergence가 아닌 consecutive ratio##만 사용 → magnitude scaling 무관 + ##recovery pattern을 자연스럽게 포착## (ratio가 1 넘어가면 weight 회복). Heuristic exponential decay와 정성적으로 다른 properties.
  • Ablation의 풍부함: Reweighting 4 variant + objective 2 variant + scaling (4B vs 14B teacher) + dynamics (entropy, mean turns)까지 multi-axis로 검증. ##개별 component의 indispensability##가 명확.
  • SLM TIR의 실용 SOTA: 0.6B AIME 2025 26.13%는 sub-billion scale에서 인상적. ##On-device deployment 가능한 size에 frontier reasoning approach## 가능성을 보임.
  • Open release: GitHub 공개로 재현·확장 가능.

한계와 의문점

  • Qwen3 단일 family: 저자도 인정. Llama/DeepSeek/Mistral SLM에서 같은 패턴이 나오는지 미검증. SOD의 효과가 Qwen3-specific일 가능성.
  • Python code interpreter 단일 tool 환경: API call, web search, GUI tool 등 ##더 복잡한 agentic setting##에서 SOD의 reweighting이 동일 효과인지 unclear. discontinuous transition 정도가 다를 수 있음.
  • $d_k$ proxy의 정밀성: $|\log\pi_\theta - \log\pi_T|$의 단순 평균이 KL divergence $\Delta_k$의 monotone proxy라는 주장(Appendix D.5)이 ##일반 distribution에서 항상 성립##하는지의 충분성 미세분석 부재. Pathological case에서는 미스매치 가능.
  • Hyperparam sensitivity: $\epsilon$ (stability), $\delta$ (max amplification)의 ##sensitivity analysis 부재##. 실제 deployment에서 task별로 tuning 필요한지 unclear.
  • Tool error만 다룸: Erroneous tool call cascade에 집중. ##Tool-supplied correct observation이지만 student가 misinterpret##하는 case의 reweighting 효과는 분석되지 않음.
  • "Recovery from earlier errors" 빈도/효과 미세분석: Three pattern 중 recovery가 얼마나 자주 발생하는지, recovery 시 정확도 향상 기여도가 정량화되지 않음.
  • Compute cost 미공개: Step-level $d_k$ 계산이 OPD forward pass에서 zero marginal cost라고 명시되었지만, ##총 학습 GPU-hour, FLOPs## 등 SOD vs OPD 비교 부재.
  • Multi-step sparse reward 외 reward 미평가: GRPO outcome reward만 사용. Process reward model, intermediate verification 등이 SOD reweighting과 어떻게 상호작용할지 미평가.
  • Teacher quality dependence: Teacher가 ##GRPO로 추가 최적화된 Qwen3-4B##. Off-the-shelf teacher (e.g., Qwen3-4B base나 Instruct)에서 SOD 작동 여부, teacher quality가 성능에 미치는 elasticity 미평가.
  • Sub-billion claim의 대조점: "0.6B가 AIME25에서 26.13% achieve"가 첫 sub-billion이라는 주장은 ##비교 대상 sub-billion 모델 명시 부재##. 후속 검증 필요.

8. Key Takeaways

  1. OPD에는 도메인-specific failure mode 존재: Text-only reasoning에서 작동하는 dense token-level supervision이 ##discontinuous state transition (TIR)에서는 systematically 위험##. Tool-using agent를 distill할 때 OPD를 vanilla 적용하지 말 것.
  2. Multiplicative ratio reweighting의 일반성: $w_k = \prod (d_u + \epsilon)/(d_{u+1}+\epsilon)$ pattern은 ##recovery##를 자연 포착하면서 magnitude scaling에 무관. ##Token-level이 아닌 step-level divergence##라는 단위 선택이 핵심. 다른 sequential learning task (multi-turn dialogue, video, RL with hierarchical reward)에 transfer 가능 패턴.
  3. Sub-billion model의 frontier reasoning 가능성: 0.6B가 AIME25 26.13% -- ##strong post-training pipeline이 있으면 small model도 frontier benchmark에 진입 가능##. VibeThinker-3B의 Compression-Coverage Hypothesis와 일관: reasoning은 작은 모델로 압축 가능.
  4. GRPO + OPD-step의 complementary 역할: GRPO가 ##trajectory-level exploration broaden## (teacher distribution 너머), step-wise OPD가 ##fine-grained guidance##. ##두 항의 co-optimization##이 신뢰 가능 SLM TIR의 표준 architecture가 될 가능성.
  5. Stronger teacher가 항상 좋지 않을 수 있음: OPD는 4B → 14B teacher 전환 시 0.6B student가 ##저하## (capacity gap이 mismatch propagation amplify). SOD reweighting은 이 효과를 reverse. ##Distillation 시 teacher selection의 단순한 "더 큰 게 좋다"가 항상 옳지 않음##.
  6. Three Distillation Pattern으로 학습 진단: Stable / Erroneous / Recovery 3 패턴 비율을 monitoring하면 ##학습이 정상인지##(stable dominant) ##corrupted state 누적 중인지## 알 수 있다. SOD의 ##reweighting trajectory를 observability tool로 활용 가능##.
  7. Theoretical proxy가 적절한 단순화: $\Delta_k$ KL divergence를 $d_k = |\log\pi - \log\pi_T|$ 단순 평균으로 근사. ##forward pass에서 zero marginal cost##라는 implementation 효율성이 design 결정의 lever. 비슷한 단순화 기법이 다른 distillation 알고리즘에서도 적용 가능 패턴.

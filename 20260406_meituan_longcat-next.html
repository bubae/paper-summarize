TL;DR

  • 문제: 현재 multimodal 시스템들은 비언어 modality를 "language-plus-auxiliary" 방식으로 취급하여 fragmented architecture를 초래하고, 이해(understanding)와 생성(generation) 간의 conflict가 발생하며, discrete visual modeling은 continuous 대비 고유한 성능 ceiling이 존재한다고 여겨진다.
  • 해결: Discrete Native Autoregression (DiNA) 패러다임을 도입하여, 모든 modality를 shared discrete token space에서 Next-Token Prediction (NTP)으로 통합한다. 핵심은 dNaViT (Discrete Native-Resolution Vision Transformer)로, Semantic-and-Aligned Encoder (SAE) + Residual Vector Quantization (RVQ)를 통해 semantic completeness를 보장하면서 information loss를 최소화한다.
  • 왜 작동하는가: SAE가 semantic completeness를 보장하고 (discriminative invariance + generative sufficiency), RVQ가 residual-of-residual 방식으로 정보 손실을 최소화하며, encoder의 residual connection이 명시적 reconstruction supervision 없이도 low-level signal을 보존하는 intrinsic information recovery를 제공한다. Modality-agnostic MoE가 학습 과정에서 자연스럽게 modality-specialized expert로 진화한다.
  • 비유: 언어가 이미 이산 토큰(단어)으로 성공적으로 모델링되듯, 시각과 음성도 이산 "단어"로 변환하면 하나의 언어 모델이 보고(seeing), 그리고(painting), 말하는(talking) 것을 동시에 수행할 수 있다. 마치 다국어 사전을 하나의 통합 사전으로 합치는 것처럼, 각 modality의 "어휘"를 공유 공간에 배치하는 것이다.

핵심 기여 (Key Contributions)

  1. DiNA (Discrete Native Autoregression) 패러다임: NTP를 language에서 native multimodality로 확장한다. Modality-specific tokenizer-detokenizer 쌍만 설계하면 decoder-only backbone이 multi-task learner로 기능하며, text, vision, audio를 하나의 autoregressive objective 하에서 처리한다.
  2. dNaViT (Discrete Native-Resolution Vision Transformer): SAE + RVQ 기반 vision tokenizer로, semantic completeness 원칙에 따라 이해와 생성 모두를 지원한다. 임의 해상도를 처리하며, 28x compression ratio로 효율적인 discrete visual representation을 제공한다.
  3. Understanding-Generation 통합: Discrete modeling에서 이해와 생성의 conflict를 효과적으로 해결한다. 실험적으로 ##생성 학습이 이해를 오히려 enhance한다는 것##을 증명하였다 (Figure 10 --- unified model이 pure-understanding model 대비 loss 차이 0.006에 불과하며, pure-generation 대비 생성 loss는 0.02 낮음).
  4. Industrial-strength 성능: A3B MoE (68.5B total, ~3B activated)에서 Qwen3-Omni-A3B-Instruct를 시각 이해에서 초과하고, Flux-dev와 T2I에서 경쟁하며, audio에서 SOTA를 달성한다. 텍스트 능력도 저하 없이 유지된다.

1. Introduction

NTP 패러다임의 성공과 multimodality 확장의 도전

Large Language Models (LLMs)는 Next-Token Prediction (NTP) 패러다임에 수렴하였으며, 이 discrete autoregressive modeling이 intelligence의 핵심 드라이버임이 입증되었다. 그러나 현실 세계의 정보는 text, vision, audio 등 다수의 modality에 걸쳐 있으며, 대부분의 기존 multimodal 시스템은 비언어 modality를 language model에 느슨하게 결합된 보조적 bolt-on component로 취급하는 language-plus-auxiliary 패러다임에 머물러 있다.

저자들은 multimodality를 language의 native linguistic extension으로 개념화할 때, 모든 modality가 하나의 shared autoregressive objective 하에서 interoperable token sequence로 표현되므로 문제가 크게 단순화된다고 주장한다. 그러나 핵심 질문이 남는다: 비언어 modality를 discrete token space 안에서 어떻게 효과적으로 표현할 것인가?

Discrete visual modeling의 dual bottleneck

Audio의 경우 자연어와 유사한 이산적 특성 덕분에 discrete autoregressive modeling이 상당한 성공을 거두었으나, vision으로의 확장은 근본적으로 다른 도전을 제기한다. Visual signal은 고차원이고 연속적이며, 유한한 codebook으로 압축하면 필연적으로 information loss가 발생한다. 저자들은 이 문제를 dual bottleneck으로 정의한다:

  1. Capacity of visual representation: Discrete representation이 원본 이미지의 semantic 정보를 충분히 담을 수 있는가? 이를 위해 semantic completeness의 중요성을 강조하며, Semantic-and-Aligned Encoders (SAE)가 강력한 foundation이 된다.
  2. Information loss from discretization: 연속 feature를 이산 codebook에 매핑할 때의 정보 손실 문제. 이를 해결하기 위해 Residual Vector Quantization (RVQ)으로 "residual의 residual"을 모델링하여 정보를 계층적으로 보존한다.

Intrinsic information recovery의 발견

흥미롭게도, 저자들은 encoder의 residual architecture가 명시적 reconstruction supervision 없이도 low-level signal 전파를 위한 latent pathway를 본질적으로 보존한다는 것을 발견하였다. 이를 intrinsic information recovery라 명명한다. 실험적으로 randomly initialized ViT-B/16이 pretrained ViT-B/16보다 reconstruction에서 우수하며 (PSNR 30.52 vs 21.86), 이는 정보 회복이 학습된 semantic이 아닌 네트워크 구조 자체의 residual connection에 기인함을 시사한다.

dNaViT 소개

이러한 통찰을 기반으로, 저자들은 Discrete Native-Resolution Vision Transformer (dNaViT)를 소개한다. dNaViT는 paired tokenization/de-tokenization을 수행하여, 이미지를 semantic completeness를 가진 discrete ID로 encode하고 동시에 reconstruction도 지원한다. 임의 해상도에서 작동하며, 최대 28x compression ratio를 달성한다. Multi-level residual token을 shared representational currency로 활용하여, discrete ID와 이미지 간의 bidirectional mapping을 가능하게 한다.

2. Methodology

Discrete autoregressive 패러다임을 multimodality로 확장하면 다음과 같은 핵심 이점이 있다: (1) Architectural Synergy --- LLM의 성숙한 optimization/scaling infrastructure 재사용, (2) Unification of Understanding and Generation --- 하나의 NTP objective로 이해와 생성을 통합, (3) Seamless Cross-Modal Interaction --- modality 간 자연스러운 상호작용, (4) Native Data Scaling and Unified Self-Supervision --- uncurated in-the-wild 데이터에서 자기지도 학습 가능.

이를 production-ready로 만들기 위해, 저자들은 세 가지 기준을 제시한다:

  • Performance Parity and Beyond: 전문 모델과 동등하거나 초과하는 성능
  • Modality Synergy Instead of Compromise: Multimodality 확장 시 language 능력 저하 없이 cross-modal synergy 달성
  • Infrastructure-Friendly Evolution: 기존 LLM 프레임워크와의 호환성, 최소한의 modality-specific inductive bias

2.2 Vision Tokenizer

2.2.1 Design Motivation: Semantic Completeness

Language tokenizer의 성공은 near-lossless discrete compression --- subword tokenization이 semantic content를 보존하면서 faithful reconstruction을 위한 충분한 구조를 유지 --- 에 기반한다. Visual tokenizer도 이와 유사한 역할을 해야 하지만, visual 정보의 본질적으로 높은 information density 때문에 어렵다.

저자들은 semantic completeness라는 원칙을 제안한다. Discrete representation $z$가 semantically complete하다 함은, 임의의 image-centric inquiry $\mathcal{Q}$에 대해 $z$에 조건화된 posterior distribution이 원본 이미지 $I$에 조건화된 것을 근사함을 의미한다:

$$\mathcal{P}(A \mid z, \mathcal{Q}) \approx \mathcal{P}(A \mid I, \mathcal{Q}) \tag{1}$$

직관적으로 말하면, discrete 표현 $z$만으로도 원본 이미지 $I$를 보고 답할 수 있는 모든 질문에 거의 동일하게 답할 수 있어야 한다는 것이다. 이 동치 조건은 두 가지 근본적 속성을 함의한다:

  • Discriminative Invariance: Discretization이 원본 이미지의 핵심 semantic attribute를 보존해야 한다. Fine-grained recognition부터 semantic reasoning까지, discrete representation $z$는 raw pixel $I$에 포함된 critical information을 유지해야 한다.
  • Generative Sufficiency: Discrete code $z$가 faithful image generation에 필요한 essential visual semantic을 캡처해야 한다. De-tokenizer $D(\cdot)$가 이미지의 structural, textural content를 재구성할 수 있어야 한다 ($I' \approx D(z)$).

2.2.2 Semantic-and-Aligned Encoder (SAE)

Language-aligned semantic encoder가 pre-quantization space로서 특히 적합하다. 그 이유는 두 가지 핵심 속성 때문이다: (i) semantic richness --- high-level concept와 fine-grained visual detail 모두를 캡처, (ii) affinity with language models --- unified discrete token space로의 seamless 통합을 가능하게 함.

SAE를 이미지 $I$를 pre-quantization representation $z_p$로 사영하는 mapping $\mathbf{z_p} = \mathcal{E}_{\text{sae}}(I)$로 정의한다. 이 representation은 다양한 image-centric query $\mathcal{Q}$에 답하는 데 필요한 정보를 보존해야 한다:

$$\mathcal{L}_{\text{SAE}} = \mathbb{E}_{(I, \mathcal{Q}, A)} \left[ -\log P(A \mid \mathbf{z_p}, \mathcal{Q}) \right] \tag{2}$$

직관적으로, SAE의 학습 목표는 이미지에 관한 어떤 질문이든 SAE 표현만으로 올바르게 답할 수 있도록 하는 것이다.

실제로 semantic completeness는 large-scale vision-language training을 통해 암묵적으로 달성된다: (i) Global Alignment --- CLIP-style contrastive learning으로 coarse cross-modal correspondence 확립, (ii) Detailed Learning --- multi-aspect language-conditioned supervision으로 fine-grained detail과 nuanced semantic 캡처. QwenViT, MoonViT, AIMv2 등 기존 vision-language model의 encoder를 strong SAE approximation으로 직접 활용할 수 있다.

2.2.3 Tokenization: Residual Vector Quantization (RVQ)

SAE가 semantically complete manifold를 제공하더라도, 유한 discrete space로의 discretization은 fidelity loss를 수반한다. 이를 해결하기 위해 Residual Vector Quantization (RVQ)을 채택한다. Single-step mapping 대신, $L$개의 cascaded codebook level로 SAE feature $\mathbf{z}$를 분해한다:

$$\mathbf{r}_0 = f_{\text{proj}}(\mathbf{z}), \quad \hat{\mathbf{q}}_l = \text{VQ}(\mathbf{r}_{l-1}), \quad \mathbf{r}_l = \mathbf{r}_{l-1} - \hat{\mathbf{q}}_l, \quad \hat{\mathbf{z}} = \sum_{l=1}^{L} \hat{\mathbf{q}}_l \tag{3}$$

직관적으로, 첫 번째 level은 "대략적인 스케치"를, 각 후속 level은 이전 level이 놓친 "디테일"을 잡아내는 방식이다. $\mathbf{r}_0$은 projected latent representation, $\hat{\mathbf{q}}_l$은 level $l$에서의 quantized feature, $\mathbf{r}_l$은 해당 level의 residual error이다.

EMA Codebook Update: Gradient descent 대신 exponential moving average (EMA)로 codebook entry를 업데이트한다. 각 codebook entry $\mathbf{e}_k$에 대해 running cluster size $N_k$와 embedding sum $\mathbf{m}_k$를 유지한다:

$$\mathbf{e}_k \leftarrow \frac{\mathbf{m}_k}{N_k} \tag{4}$$

Laplace smoothing으로 수치 안정성을 확보하고, 비활성 entry ($N_k < 1$)는 현재 batch에서 re-initialize하여 codebook utilization을 유지한다.

Quantization Objective: Projection function $f_{\text{proj}}$에 supervision을 제공하기 위해, quantized embedding에서 pre-quantization semantic feature를 재구성하는 lightweight decoder를 도입한다:

$$\mathcal{L}_{\text{quant}} = \lambda_c \mathcal{L}_{\text{commit}} + \lambda_s \mathcal{L}_{\text{semantic}} \tag{5}$$

$\mathcal{L}_{\text{commit}}$은 projected representation $\mathbf{r}_0$을 assigned codebook entry에 가깝게 유지하는 standard commitment loss이고, $\mathcal{L}_{\text{semantic}}$은 feature cosine similarity를 활용한 semantic reconstruction loss이다.

전체 tokenization pipeline은 3단계로 구성된다: (1) SAE를 통한 semantically aligned feature 추출 → (2) RVQ with EMA-updated codebook을 통한 hierarchical discretization → (3) Autoregressive modeling을 위한 semantic token 생성.

2.2.4 De-tokenization

Tokenization 학습 후, SAE encoder와 codebook은 frozen된다. De-tokenizer는 pixel decoder를 중심으로 구성되며, discrete code embedding만으로 이미지를 재구성하는 Vision Transformer 기반의 400M-parameter 모델이다:

$$\mathcal{L}_{\text{dec}} = \lambda_1 \mathcal{L}_{\text{pixel}} + \lambda_2 \mathcal{L}_{\text{percep}} + \lambda_3 \mathcal{L}_{\text{align}} \tag{6}$$

$\mathcal{L}_{\text{pixel}}$은 pixel-level reconstruction loss, $\mathcal{L}_{\text{percep}}$은 perceptual loss, $\mathcal{L}_{\text{align}}$은 alignment loss이다.

Pixel decoder만으로도 coherent spatial layout, object structure, semantic content를 복원할 수 있으나, 결과 이미지가 지나치게 smooth하고 high-frequency detail이 부족하다. 이를 보완하기 위해 flow-matching objective $\mathcal{L}_{\text{flow}}$로 학습된 lightweight image refiner를 도입한다. Refiner는 pixel decoder의 reconstruction과 projected discrete code embedding을 conditioning으로 받아, fine-grained texture를 향상시킨다. Refiner는 OmniGen2에서 초기화된다.

Intrinsic Information Recovery: Residual Connection의 역할

Semantic completeness를 달성하려면 unified representation $\mathbf{z_p}$가 semantic comprehension뿐만 아니라 generative sufficiency도 가져야 한다. 저자들은 기존 연구들이 주장하는 "semantically enriched latent prior"에 의한 recoverability 대신, encoder의 본질적 architectural property인 residual connection에 주목한다.

Encoder를 $L$개의 residual block 시퀀스로 표현하면, 최종 latent representation $\mathbf{z}_p$는:

$$\mathbf{z}_p = \mathbf{x}_0 + \sum_{l=1}^{L} \mathcal{F}_l(\mathbf{x}_{l-1}) = \underbrace{\mathbf{x}_0 + \mathcal{F}_1(\mathbf{x}_0) + \cdots}_{\text{pixel-level} \longrightarrow \text{semantic-level}} + \mathcal{F}_L(\mathbf{x}_{L-1}), \quad \text{where} \quad \mathbf{x}_l = \underbrace{\mathbf{x}_{l-1} + \mathcal{F}_l(\mathbf{x}_{l-1})}_{\text{residual connection}} \tag{7}$$

직관적으로, identity mapping $\mathbf{x}_{l-1}$이 earlier layer의 fine-grained visual signal이 higher-level semantic abstraction에 의해 덮어쓰이지 않도록 보장한다. 따라서 $\mathbf{z}_p$는 원본 이미지 $I$와 높은 mutual information을 유지하는 representation이 된다.

2.2.5 dNaViT: Discrete Native-Resolution Vision Transformer

위의 semantic completeness와 discrete quantization 원칙을 기반으로, dNaViT를 소개한다. Language tokenizer와 유사한 역할을 하며, 이미지가 semantically meaningful discrete symbol의 sequence로 표현되어 autoregressive modeling과 직접 호환된다.

Native Resolution: 기존의 fixed-resolution input 대신, dNaViT는 이미지를 native resolution에서 직접 처리한다. Resizing, cropping, padding에 의한 information loss와 spatial distortion을 피하며, 임의 image scale에서 visual detail을 충실히 보존한다.

Hierarchical Discrete Tokens: Multi-level discrete token을 구성하여, 여러 semantic level에 걸친 정보를 encoding한다. 이 계층적 구조는 global structure와 local detail을 모두 unified discrete representation 안에서 보존한다.

Native Visual Representation: dNaViT의 핵심 혁신은 visual codebook embedding이 randomly initialized되어 LLM과 함께 co-evolve한다는 점이다. Semantic content가 tokenizer 내부에 고정되는 것이 아니라, LLM의 visual embedding space에 internalize된다. 이를 통해 모델은 externally defined quantized ID가 아닌, learned model-internal visual semantic에 기반한 "native language" for vision을 개발한다.

Additive Encoding: Autoregressive modeling 시, multi-layer token에 대해 additive encoding을 사용하여 exponential representation space를 확보하면서도 single autoregressive step의 computational efficiency를 유지한다.

DepthTransformer for Decoding: Multi-level token의 efficient decoding을 위해 task-aware DepthTransformer를 사용한다.

2.3 Audio Tokenizer

Audio tokenizer는 continuous speech를 discrete token으로 변환하며, semantic과 acoustic information 모두를 보존한다.

Audio Tokenizer 파이프라인: [입력: Mel Spectrogram] ↓ [Whisper Encoder] — audio feature 추출 ↓ [4× Downsample] — temporal 해상도 감소 ↓ [8-layer RVQ] — codebook sizes: 8k, 4k, 2k, 1k, 1k, 1k, 1k, 1k ↓ [Discrete Audio Tokens] ↓ (두 갈래) ├─→ [Audio Decoder] → [Flow Matching Decoder] → [Vocoder] → Mel Spectrogram → Audio └─→ [Pretrained LLM (Qwen3-1.7B)] → Text (audio understanding)

Audio token은 두 개의 parallel branch로 전달된다:

  • Branch 1 (Understanding): Frozen pretrained LLM (Qwen3-1.7B)에 투입되어 audio understanding을 수행한다. LLM의 textual embedding space와 align되며, LLM architecture/vocabulary 간 transferable하다.
  • Branch 2 (Reconstruction): Encoder와 symmetric architecture의 decoder로 coarse Mel spectrogram을 재구성한 뒤, flow-matching model이 refine하고, vocoder (HiFi-GAN)가 최종 waveform을 생성한다.

학습 목표:

$$\mathcal{L}_{\text{audio}} = \lambda_1 \mathcal{L}_{\text{recon}} + \lambda_2 \mathcal{L}_{\text{commit}} + \lambda_3 \mathcal{L}_{\text{llm}} \tag{8}$$

직관적으로, $\mathcal{L}_{\text{recon}}$은 소리를 얼마나 잘 복원하는지, $\mathcal{L}_{\text{commit}}$은 codebook entry에 얼마나 가까운지, $\mathcal{L}_{\text{llm}}$은 audio token이 언어 모델의 이해력을 얼마나 잘 지원하는지를 나타낸다.

3-stage training:

  1. Stage 1: Decoder Warm-up --- Encoder (Whisper-large-v3)와 LLM (Qwen3-1.7B) frozen, decoder만 Mel spectrogram reconstruction task로 학습
  2. Stage 2: Semantic-Acoustic Joint Training --- LLM과 flow-matching 제외, 나머지 모든 모듈 업데이트. 세 가지 loss 모두 활성화. RVQ 8-layer (codebook sizes: 8k, 4k, 2k, 1k, 1k, 1k, 1k, 1k)
  3. Stage 3: Decoder Fine-tuning --- 24kHz high-quality audio로 decoder reconstruction 향상. Decoder를 Diffusion Transformer (DiT) 모듈로 재구성하여 RVQ artifact를 denoise

2.4 Language Model Backbone

LongCat-Flash-Lite A3B를 decoder-only backbone으로 채택한다. 68.5B 총 파라미터에 평균 3B 활성화 (2.9B~4.5B 범위), Zero-Expert and Shortcut MoE architecture를 사용한다. Modality-agnostic 설계가 핵심이며, modality-aware MoE, 3D RoPE, bidirectional attention 등의 modality-specific branching을 일절 사용하지 않는다. 이를 통해 모든 modality의 token이 unified latent space에서 동일하게 처리된다.

2.5 Multimodality Component

2.5.1 End-to-End Multimodal Embedding

기존 multimodal 접근들이 continuous visual feature를 language model의 embedding space에 project하는 것과 달리, LongCat-Next의 모든 multimodal embedding은 randomly initialized되어 native model과 jointly 학습된다.

  • Visual embedding: Codebook 크기 $8 \times 16{,}384$, multi-level token은 multi-level summation으로 결합. Level 간 embedding은 공유되지 않아, 각 level이 complementary information을 캡처한다.
  • Audio embedding: 같은 설계를 따르되, decreasing codebook size를 사용한다.

Pre-quantization feature는 RVQ process에서 discrete clustering assignment를 설정하는 데만 사용되며, embedding value 자체를 직접 결정하지 않는다. Embedding은 end-to-end로 학습되어 multimodal understanding과 generation 모두를 지원한다.

2.5.2 Multimodality Head

Multi-level supervision scheme을 채택한다. Language model은 single-step autoregressive prediction을 수행하고, multi-level token은 parallel로 decode된다:

  • LLM head: Standard MLP
  • Multimodal token head: Task-aware DepthTransformer가 multi-level output을 생성하여 structured visual/audio information을 복원한다.

2.5.3 Internal Linguistic Guidance

Speech language model에서 speech interaction이 text와 동일한 linguistic capability를 상속할 수 있는지가 중요한 과제이다. Moshi의 접근을 참고하여, speech generation 과정에 internal linguistic guidance를 제공한다.

Special tokens: AS (Audio Start), AE (Audio End), TE (Text End) --- 각각 audio segment의 시작/끝과 text segment의 끝을 표시한다.

두 가지 생성 전략:

  1. Parallel Generation: 각 decoding step에서 text와 audio token을 동시 생성하되, 첫 audio token의 생성을 지정된 step만큼 의도적으로 delay한다. Response latency를 제거하여 full-duplex modeling에 적합하다.
  2. Serial Generation: Text segment를 먼저 생성한 후, 해당 audio segment를 생성한다. 각 step에서 single modality만 예측하므로 modality representation 간 conflict를 회피하고, 높은 linguistic quality를 보장한다.

Random delay training paradigm: Text-guided audio modality에서 각 aligned text-audio segment의 delay를 1부터 text segment 길이까지 랜덤하게 선택한다. 이를 통해 모델이 arbitrary delay step에서 text-audio alignment을 학습하며, serial/parallel generation을 unified framework의 두 extreme case로 통합한다.

3. Main Experiments

LongCat-Next는 text, vision, speech를 동시에 처리할 수 있는 unified omni-modal 시스템이다. 주요 비교 대상은 Qwen3-Omni-A3B-Instruct (현재 SOTA omni model)이며, 시각 이해를 독립적으로 평가하기 위해 Qwen3-VL-A3B-Instruct도 포함한다. Audio domain에서는 MiMo-Audio, Kimi-Audio, Step-Audio-2-mini와도 비교한다.

3.2 Experimental Analysis of Methodology

핵심 component의 effectiveness를 분석하는 ablation study들이다. Computational cost 제약으로 일부 실험은 Qwen-7B backbone에서 수행되었다.

3.2.1 Bridging the Understanding Gap Between Discrete and Continuous Modeling

실험 의도: Discrete representation이 continuous representation과 비교하여 어느 정도의 성능 gap이 있으며, 이를 어떻게 줄일 수 있는지 규명한다.

설정: Language backbone과 vision encoder를 frozen하고, discrete dNaViT vs continuous NaViT만 다르게 하여 pre-alignment stage에서 captioning loss를 proxy로 정보 손실을 측정한다.

세 가지 핵심 발견 (Figure 9):

  1. 초기 gap: Continuous feature의 initial training loss가 significantly 낮다. Continuous feature가 초기 align에 inherently 더 유리하다.

3.2.2 Information Recovery Analysis

실험 의도: Reconstruction capability가 학습된 semantic에 기인하는지, 아니면 network architecture의 intrinsic property인지 규명한다.

다양한 frozen vision encoder를 lightweight ViT-based pixel decoder와 쌍으로 테스트하였다. 결과는 Table 6 (앞서 제시)에서 확인한 바와 같다:

  • Residual pathway가 있는 encoder (ViT 계열)는 explicit reconstruction supervision 없이도 non-trivial reconstruction capability를 보인다.
  • Random ViT-B/16 > Pretrained ViT-B/16: PSNR 30.52 vs 21.86. Random ViT의 출력은 noise-like signal로 decoder가 denoise하기 쉬우며, residual pathway가 pixel-level information을 보존한다.
  • QwenViT (w/o merger): Merger 없으면 comparable reconstruction이나, merger 도입 시 aggressive downsampling (14x → 28x)으로 degradation 발생.

이 결과는 SAE-style encoder (QwenViT 등)가 coarse image-level structure를 복원할 수 있지만, fine-grained high-frequency detail 재구성에는 덜 effective함을 보여준다. 이는 pixel-level과 semantic-level information 간의 trade-off에 대한 새로운 관점을 제공한다.

3.2.3 Conflict between Understanding and Generation

실험 의도: DiNA framework 하에서 이해와 생성이 conflicting하는지, synergistic한지 규명한다.

설정: 동일한 model checkpoint에서 출발하여 세 모델을 학습:

  • Pure-Und.: 100B token의 understanding data만으로 학습
  • Pure-Gen.: 100B token의 generation data만으로 학습
  • Unified: 1:1 mixture로 50B understanding + 50B generation (총 100B)

결과 (Figure 10):

  • Unified model의 understanding loss = Pure-Und. 대비 단 0.006 높음 (marginal)
  • Unified model의 generation loss = Pure-Gen. 대비 0.02 낮음 (significant improvement)

##이는 generation 학습이 understanding을 저해하지 않을 뿐 아니라, understanding이 generation을 오히려 actively enhance함을 증명한다## (질문 4에 대한 답). 이해 task가 풍부한 semantic representation을 제공하여 generation의 semantic planning을 강화하는 것으로 해석된다.

  • Qwen3.5: Data-native training으로 partial cross-modal alignment
  • LongCat-Next: ##Text-visual embedding이 interwoven (교차 배치)## --- 서로 다른 modality의 feature가 multilingual expression처럼 동일한 underlying concept의 다른 표현으로 기능한다 (질문 5에 대한 답)

추가적으로, frozen visual tokenizer의 quantized mapping이 LongCat architecture에 통합 시 negligible performance degradation을 보여, discrete semantic space가 language representation space와 inherently aligned되어 있음을 나타낸다.

4. Implementation Details

4.1 Training Stage

DiNA의 최적화는 두 phase로 구성된다: (1) Tokenizer Training --- modality-specific tokenizer/detokenizer의 독립 학습, (2) Native Multimodality Training --- unified multimodal 학습.

4.1.1 Visual Tokenizer

SAE로 Qwen2.5-ViT (28x spatial compression ratio)를 직접 채택하며, 별도의 SAE training은 수행하지 않는다.

Stage 1: Visual Tokenization

  • SAE feature의 continuous manifold를 discrete latent space로 매핑
  • RVQ 사용, 8-stage (8-level residual quantization)
  • Fixed-resolution → any-resolution으로 2-phase training
  • Maximum training sequence length: 8,192 tokens
  • Training corpus: ~50M images (LAION, COYO, DataComp, TextAtlas + in-house visual understanding + synthetic data from MidJourney)
  • Maximum image resolution: 1,736 x 1,736
  • Variable-length FlashAttention으로 arbitrary resolution 처리

Stage 2: Visual De-tokenization

  • 400M-parameter ViT-based pixel decoder, trained from scratch
  • MLP-based patch unmerger → transformer layers with 2D RoPE → linear head to pixel space
  • Image refiner: OmniGen2에서 초기화, flow-matching loss로 학습
  • Conditioning: (1) pixel decoder reconstruction + noise latent (spatial guidance), (2) projected discrete code embedding (semantic conditioning)
  • Training data: Stage 1과 동일 corpus + SAM-1B, RenderedText, IDL, high-resolution in-house images
  • SAE encoder와 codebook frozen, de-tokenizer는 native resolution까지 학습

4.1.2 Audio Tokenizer

  • Encoder: Whisper-large-v3 + Qwen3-1.7B LLM (tokenizer training 시에만 사용, 이후 폐기)
  • Training corpus: ~2.5M hours (web-collected Chinese/English speech + curated multilingual/dialectal ASR + synthetic speech + music/sound captioning)

5. Infrastructure

5.1 VHalf-based Pipeline Parallelism

Multimodal model의 computational workload는 inherently heterogeneous하다. Embedding layer와 modality-specific loss module (DepthTransformer 등)의 execution time이 core LLM transformer layer와 significantly 다르며, token distribution에 따라 동적으로 fluctuate한다. Naive linear pipeline parallelism은 pronounced inter-stage load imbalance와 excessive P2P communication overhead를 초래한다.

이를 해결하기 위해 V-Half 기반 pipeline parallel architecture를 제안한다:

  1. V-shaped Schedule: Linear assignment 대신, 첫 computational stage (embedding layer)와 마지막 computational stage (modality-specific loss module)를 동일한 physical device에 co-locate한다.
  2. Shared Buffer: V-shaped pipeline의 first stage에 shared buffer를 구성하여, modality loss module이 embedding lookup 후 RVQ multi-level token에 해당하는 hidden state에 직접 접근 가능하게 한다.

추가 최적화:

  • Co-locating Embedding and Modality Loss: Embedding layer와 modality loss module을 independent chunk으로 동일 pipeline stage에 배치. Profile-guided assignment으로 combined latency가 ideal uniform chunk latency 내에 유지되도록 보장.
  • Decoupling LLM Head: V-shape의 anchor device overloading 방지를 위해, LLM head를 소수의 LLM transformer layer와 함께 별도 pipeline stage로 분리.
  • Adaptive LLM Layer Distribution: 나머지 LLM transformer layer를 나머지 chunk에 균등 분배.

효과:

  • Effective Bubble Mitigation: Heterogeneous workload의 dynamic overhead가 schedule에 의해 흡수, near-perfect load balance 달성.
  • Elimination of Cross-Stage Communication: Embedding layer와 modality loss module이 동일 device에 co-locate되어, forward activation과 backward gradient 전송이 zero-copy intra-device memory access로 해결된다.

6. Advantages and Future Work

6.1 Reinforcement Learning

Discrete representation은 RL에서 inherent advantage를 제공한다. Discrete visual latent space가 자연스럽게 action space로 기능하며, GRPO 같은 기존 language model RL 방법과 seamless 호환이 가능하다. Flow model처럼 ODE-to-SDE 변환이 필요 없이 finite Markov Decision Process (MDP) 구조를 유지한다.

RL for Text-to-Image Generation

Policy가 input prompt로부터 discrete token sequence를 생성하고, frozen decoder가 이미지로 변환한 뒤, multi-dimensional reward model이 scalar reward를 계산한다. RVQ의 multi-level 특성을 고려한 modified GRPO objective:

$$r_t^{(l)} = \frac{\pi_\theta^{(l)}(a_t^{(l)} | s_t)}{\pi_{\text{old}}^{(l)}(a_t^{(l)} | s_t)} \tag{9}$$ $$\mathcal{L}_{\text{GRPO}} = \mathbb{E}\left[\sum_{l=1}^{L} w_l \cdot \min\left(r_t^{(l)} \cdot \hat{A}_t, \; \text{clip}(r_t^{(l)}, 1-\epsilon, 1+\epsilon) \cdot \hat{A}_t\right)\right] \tag{10}$$

직관적으로, 각 RVQ level $l$에 대해 독립적인 probability ratio $r_t^{(l)}$을 계산하고, level별 weight $w_l$로 가중 합산하여 모든 level의 기여를 균형있게 반영한다.

4가지 reward dimension:

  1. Comprehensive Capability Enhancement: Object count, color accuracy, spatial positioning, attribute consistency 평가
  2. OCR Capability Enhancement: GOT-OCR 2.0으로 rendered text의 edit distance 측정. 2~20 texts, bucket sampling
  3. Text-Image Semantic Alignment: VLM-based reward model로 prompt-image conceptual alignment 평가
  4. Image Quality Assessment: HPS, Aesthetic Score, Unified Reward로 subjective aesthetic + objective quality 측정

RL for Image Understanding

Image understanding에 RL을 적용할 때, entropy explosion이라는 심각한 문제가 발생한다. Training이 진행될수록 policy entropy와 training-inference mismatch가 동시에 증가하여, noisy/garbled token이 rollout에 혼입되고 이것이 다시 sampling probability를 높이는 positive feedback loop을 형성한다.

핵심 원인: 소수의 anomalous token에서 training-inference probability divergence가 극심 (예: sampling policy에서 0.4 이상이나 training policy에서는 $10^{-2}$ 수준). 이러한 token이 이후 generation을 오염시킨다.

해결: 두 가지 sequence-level filter:

  1. Entropy-based Filter: 각 minibatch에서 sequence-wise entropy의 mean $\mu_H$와 variance $\sigma_H$를 계산. $H_{\text{seq}} > \mu_H + n\sigma_H$인 sequence를 outlier로 필터링.
  2. Training-Inference Probability Difference Filter: 각 rollout에서 per-token probability difference를 모니터링. 임의 token에서 $|\bar{\pi}_{\text{sampler}}(a_{t'}|s_{t'}) - \bar{\pi}_{\text{actor}}(a_{t'}|s_{t'})| > \delta$이면, 해당 sequence 전체를 discard.

통합 objective:

$$\mathcal{L}_{\text{GRPO-Filtered}} = \mathbb{E}_{(s_t, a_t) \sim \mathcal{D}}\left[\underbrace{\mathbb{I}\{H_{\text{seq}}(s_t) \leq \mu_H + n\sigma_H\}}_{\text{(1) Entropy Filter}} \cdot \underbrace{\mathbb{I}\left\{\max_{t' \in \mathcal{T}_y} |\bar{\pi}_{\text{sampler}}(a_{t'}|s_{t'}) - \bar{\pi}_{\text{actor}}(a_{t'}|s_{t'})| \leq \delta\right\}}_{\text{(2) Prob. Diff. Filter}}\right.$$ $$\left.\cdot \min\left(r_t(\theta) \cdot \hat{A}_t, \; \text{clip}(r_t(\theta), 1-\epsilon, 1+\epsilon) \cdot \hat{A}_t\right)\right] \tag{11}$$

직관적으로, $\mathbb{I}\{\cdot\}$는 hard binary mask로 작동하여, 두 filter를 모두 통과한 sequence만 gradient update에 기여한다. Token-level이 아닌 sequence-level에서 필터링하므로, 소수의 anomalous token이 전체 sequence를 오염시키는 문제를 근본적으로 차단한다.

6.2 Discussion and Future Work

  • Vision Tokenizer: 현재 버전은 image understanding/generation에 집중하며, tokenizer가 fully optimized되지 않았다. De-tokenizer는 semantic decoding consistency에 초점을 두어 pixel fidelity보다 semantic 정확성을 우선시한다.
  • Beyond Understanding and Generation: 현재는 image-to-text와 text-to-image에 집중하지만, 진정한 native multimodal system은 any-to-any generation과 interleaved multimodal reasoning으로 확장되어야 한다.
  • Data Scaling and Representation Learning: Multimodality가 language만으로는 불가능한 complementary signal을 제공할 수 있는지가 핵심 미해결 질문이다. 기존 perceptual pretraining paradigm이 discrete modeling에서 directly translate되지 않으며, data, pretraining objective, discretization strategy의 co-design이 필요하다.

7. Conclusion

LongCat-Next를 통해 language-style discrete autoregressive modeling이 text를 넘어 vision과 audio까지 자연스럽게 확장될 수 있음을 탐색하였다. 핵심 결론은: 신중하게 설계된 tokenizer와 training strategy가 있다면, continuous perceptual signal을 효과적으로 discretize하면서도 강력한 capability를 유지할 수 있으며, discrete modeling에 inherent performance ceiling이 존재하지 않는다는 것이다.

Appendix 주요 내용

9.1 Training-Inference Mismatch Analysis in RL

RL training 중 sequence collapse가 빈번하게 발생한다. 소수의 token에서 training-inference probability divergence가 극심하여 (sampling policy 0.4 vs training policy $10^{-2}$), 나머지 정상 token의 평균 difference ($\sim 10^{-3}$)와 극적으로 다르다. 이러한 anomalous token이 catastrophic effect를 유발한다.

Truncated Importance Sampling (TIS), Multiple Importance Sampling (MIS) 등 기존 importance sampling correction은 onset을 몇 step 지연시킬 뿐 근본적 해결이 안 된다. Sequence-level correction method도 소수 anomalous token의 extreme divergence가 다수 정상 token에 의해 평균화되어 실패한다. 따라서 per-token absolute probability difference에 threshold $\delta$를 직접 적용하는 direct intervention이 가장 효과적이다.

9.3 Discrete Quantization Strategy

Feature reconstruction loss를 proxy로 discrete quantization 전략을 비교한다:

  • 2-stage RVQ가 vanilla VQ를 slightly outperform
  • 8-stage RVQ로 scale up 시 feature reconstruction loss가 significantly 감소
  • RVQ의 residual mechanism과 compositionality가 minimal information loss를 위해 essential
  • 8-stage RVQ가 과도한 computational overhead 없이 충분히 낮은 information loss를 달성 → default setting으로 채택

9.4 Vision Understanding and Generation under DiNA

DiNA 하에서 vision understanding과 generation의 통합 방식:

  • Understanding: Visual signal → hierarchical discrete tokens → additive sum → LLM input. 모든 level의 정보가 fully utilized.
  • Generation: LLM hidden representation → DepthTransformer → multi-level tokens. Single autoregressive step에서 parallel decoding으로 efficient 생성.

9.6 The Analysis of Visual De-tokenizer

ViT pixel decoder만으로는 semantic content를 재구성할 수 있지만, frozen SAE encoder의 한계로 high-frequency/fine-grained detail은 부족하다. Flow-matching refiner가 detail restoration에 critical role을 수행하여, original image를 semantic level에서 faithfully recover한다. LLM autoregression에서 predicted discrete token은 layout과 structural element 같은 semantic content를 inherently encode하며, 이것이 OCR task에서의 superior performance를 설명한다.

비판적 분석 (Critical Analysis)

강점

  1. 통합된 철학의 일관성: 모든 modality를 discrete token으로 통일한다는 단일 원칙을 architecture, training, inference 전반에 걸쳐 일관되게 적용한다. Modality-specific branching을 배제한 것은 conceptual elegance를 넘어 실질적 engineering 이점 (infrastructure 호환성, 유지보수 용이성)을 제공한다.
  2. Semantic completeness라는 원칙의 명확성: Eq.1의 정의가 vision tokenizer 설계의 모든 결정을 guide하는 통일된 원칙으로 기능한다. 이는 ad hoc한 설계 결정의 나열이 아닌 principled approach이다.
  3. Intrinsic information recovery 발견의 참신성: Random ViT > Pretrained ViT in reconstruction이라는 발견은 counter-intuitive하면서도 compelling한 증거를 제시한다. Residual architecture의 구조적 속성이 information recovery를 보장한다는 통찰은 향후 tokenizer 설계에 중요한 가이드가 된다.
  4. 광범위한 실험: Visual understanding, generation, audio, text를 모두 커버하며, 각 영역에서 competitive한 결과를 다수의 벤치마크로 검증한다. Ablation도 충실하다.

약점 및 우려 사항

  1. 재현성 우려: 68.5B 모델 + 2T token 학습은 대부분의 연구 그룹에서 재현 불가능하다. Ablation이 Qwen-7B에서 수행된 것도 full-scale 결과와의 직접적 연관성에 의문을 제기한다.
  2. 비교의 공정성: Table에서 *로 표시된 "public report" 수치가 다수이며, 동일 evaluation protocol 여부가 불명확하다. 특히 Qwen3-Omni와의 비교에서, 해당 모델의 공식 수치와 자체 평가 수치가 혼재되어 있다.
  3. 표준편차/신뢰 구간 부재: 주요 결과에 standard deviation이 보고되지 않는다 (Table 6의 reconstruction 실험 제외). Multiple seed 반복 여부가 불명확하다.
  4. SAE 자체의 학습 비용 미포함: Qwen2.5-ViT를 SAE로 "직접 채택"한다고 하지만, 이 encoder의 학습에 소요된 enormous compute cost는 LongCat-Next의 비용에 포함되지 않는다. 이는 total compute cost를 과소평가하게 만든다.
  5. De-tokenizer 한계의 솔직한 인정과 미해결: 논문이 de-tokenizer의 pixel fidelity가 semantic consistency에 비해 suboptimal함을 인정하지만, 이것이 generation quality에 미치는 영향이 FID 등의 정량적 지표로 충분히 분석되지 않았다.
  6. Any-to-any generation 미지원: 현재 image-to-text와 text-to-image만 지원하며, 진정한 omni-modal generation (image+audio 동시 생성, multi-turn grounded dialogue 등)은 future work으로 남겨둔다.
  7. Compute cost 분석 부재: Training FLOPs, inference latency, throughput 등의 computational efficiency 분석이 없다. Specialized model 대비 unified model의 practical trade-off가 불명확하다.

  • 공정한 baseline 비교: 대체로 유사 parameter scale (A3B)과 비교하나, *로 표시된 외부 수치의 evaluation 조건이 불명확 (부분적)

  • Compute cost 대비 실용성: 68.5B (3B activated) MoE는 inference 효율적이나, training cost (2T tokens)는 막대 (미분석)

Key Takeaways

  1. Discrete modeling에 inherent performance ceiling은 없다: 충분한 data scaling과 적절한 tokenizer 설계 (SAE + RVQ)로, discrete representation이 continuous와 ~1% margin까지 수렴한다. 이는 "discrete = inferior" 통념을 반박하는 중요한 실증이다.
  2. Understanding이 Generation을 enhance한다: DiNA framework 하에서 unified training의 generation loss가 pure-generation보다 0.02 낮은 반면, understanding loss 증가는 0.006에 불과하다. 이는 multimodal unified model 설계의 근거를 제공한다.
  3. Modality-agnostic MoE의 자연스러운 specialization: Explicit modality-specific routing 없이도, MoE expert가 training 과정에서 modality-specialized role을 자연스럽게 발전시킨다. 이는 modality-specific branching이 불필요할 수 있음을 시사한다.
  4. Intrinsic information recovery via residual connection: Semantic encoder의 residual architecture가 explicit reconstruction supervision 없이도 low-level signal을 보존한다는 발견은, 향후 visual tokenizer 설계 시 encoder selection의 중요한 기준을 제시한다.
  5. RL에서의 entropy explosion 해결: Discrete multimodal RL에서 training-inference mismatch로 인한 entropy explosion이 심각한 문제이며, sequence-level entropy + probability difference filter가 효과적 해법이다. 이 filtering 전략은 다른 discrete RL 시나리오에도 적용 가능하다.

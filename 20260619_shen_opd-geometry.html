3.1 Parameter-Space Diagnostics

$W_0$ = initialization, $W_+$ = post-training нӣ„ weight, $\Delta W = W_+ - W_0$.

Update sparsity

bf16-visible update convention: scalar weight $w_i, \hat{w}_i$к°Җ лӢӨмқҢ л§ҢмЎұн•ҳл©ҙ unchangedлЎң к°„мЈј:

$$|\hat{w}_i - w_i| \leq \eta \max(|w_i|, |\hat{w}_i|), \quad \eta = 10^{-3}$$

bf16-aware update sparsity:

$$\text{sp}(W_0, W_+) = 1 - \frac{1}{n}\sum_{i,j} \mathbf{1}[W_{+, ij} \not\approx_\eta W_{0, ij}] \tag{1}$$

нҒ° к°’ = fewer visible weight changes.

Principal-angle rotation

Dominant singular subspaceмқҳ rotationмқ„ ##principal angle## (Eq.2)лЎң мёЎм •:

$$\cos\theta_i(U) = \sigma_i(U_{0,k}^\top U_{+,k}), \quad \cos\theta_i(V) = \sigma_i(V_{0,k}^\top V_{+,k}) \tag{2}$$

м—¬кё°м„ң $U_{\cdot, k}$, $V_{\cdot, k}$лҠ” top-$k$ left/right singular vector. ##мһ‘мқҖ angle = pretrained dominant subspace к°•н•ң ліҙмЎҙ##. м§ҒкҙҖм ҒмңјлЎң л§җн•ҳл©ҙ, $W_0 = U \Sigma V^\top$мқҳ мЈјм¶•мқҙ н•ҷмҠө нӣ„ м–јл§ҲлӮҳ нҡҢм „н–ҲлҠ”м§ҖлҘј 1лӢЁмң„лЎң мёЎм •.

Spectral drift

Normalized Spectral Shift (NSS):

$$\text{NSS}(W_0, W_+) = \frac{\|\sigma(W_+) - \sigma(W_0)\|_2}{\|\sigma(W_0)\|_2} \tag{3}$$

$\sigma(\cdot)$лҠ” descending singular value. мһ‘мқҖ NSS = stronger spectral preservation.

Update-mask overlap

bf16 update mask $M = \{(i,j): \text{bf16}(W_+)_{ij} \neq \text{bf16}(W_0)_{ij}\}$.

  • $M_{\text{princ}}$ = $W_0$мқҳ rank-$k$ SVD reconstructionм—җм„ң ##top-$\alpha$ fraction by magnitude## (high-curvature region proxy)
  • $M_{\text{low}}$ = bottom-$\alpha$ fraction of entries by $|W_0|$ (low-magnitude region)
  • Overlap: $\text{Overlap}(M_\star, M) = |M_\star \cap M| / |M|$

Random baseline = $\alpha$. Sub-random with $M_{\text{princ}}$ = depletion from principal weight, super-random with $M_{\text{low}}$ = concentration in low-magnitude region.

OPDлҠ” RLVRмқҳ ##on-policy form мң м§Җ##н•ҳлҗҳ ##scalar credit signalмқ„ dense token-level supervisionмңјлЎң лҢҖмІҙ##.

Local quadratic budget н•ҳм—җм„ң update norm $\|\Delta W\|_F \leq \sqrt{2\delta_W / \mu_W}$ (Eq.5). м°ЁмқҙлҠ” budget мӮ¬мҡ© л°©мӢқ -- RLVRмқҖ sequence-level reward signalлЎң, OPDлҠ” teacher-token distributionмңјлЎң ##denser descent direction##мқ„ anchored region лӮҙм—җм„ң м ңкіө вҶ’ ##larger effective update than RLVR while remaining far below SFT-style unconstrained rewriting##.

Gate II (model geometry): Bounded updateк°Җ м—¬м „нһҲ pretrained model geometryм—җ мқҳн•ҙ shapeлҗЁ. OPDмқҳ relaxationмқҖ update covarianceм—җм„ң лӮҳнғҖлӮЁ:

$$\mathbb{E}[gg^\top] = \sum_{t,t'} \mathbb{E}[a_t a_{t'} \phi_t \phi_{t'}^\top] \tag{6}$$

RLVRмқҖ лӘЁл“  token scoreлҘј ##shared scalar##лЎң couple, OPDлҠ” ##heterogeneous token coefficients##лЎң broaden.

Gate III (precision realization): bf16 realization unchanged. мўҢн‘ңк°Җ visibly updatedк°Җ лҗҳл Өл©ҙ $|\Delta W_{ij}| \gtrsim \frac{1}{2}\text{ULP}_{\text{bf16}}(W_{0,ij})$ (Eq.7). OPDмқҳ larger geometry-constrained updateк°Җ ##лҚ” л§ҺмқҖ мўҢн‘ңк°Җ bf16 realization thresholdлҘј нҶөкіј##н•ҳкІҢ л§Ңл“Өм–ҙ sparsity вҶ“, off-principal bias мң м§Җ.

Implication: OPDлҠ” relaxed off-principal regimeмқ„ м°Ём§Җ -- ##RLVR-like geometry-steered##, к·ёлҹ¬лӮҳ ##SFT-like dense teacher supervisionмңјлЎң less selective##. ##OPD recipeлҠ” token-level supervision densityліҙлӢӨ update geometryлҘј regulateн•ҙм•ј н•Ё.##

4. Subspace Locking in OPD

RQ2: OPDмқҳ endpoint propertyлҝҗ м•„лӢҲлқј ##training trajectory##лҸ„ distinctн•ңк°Җ? Cumulative update $\Delta W_t = W_t - W_0$лҘј checkpointлі„лЎң м¶”м Ғ.

4.1 A Low-Dimensional Update Band (Figure 4)

к°Ғ matrixм—җ лҢҖн•ҙ ##stable rank## (Eq.8):

$$\text{srank}(\Delta W_t) = \frac{\|\Delta W_t\|_F^2}{\|\Delta W_t\|_{\text{op}}^2} \tag{8}$$

Algebraic rankмҷҖ лӢ¬лҰ¬ ##scale-aware##, "few dominant singular directionмқҙ update energyлҘј carry"н•ҳлҠ” м •лҸ„ мёЎм •.

кҙҖм°°:

  • Figure 4(a) Stable rank: ##OPDлҠ” narrow band (~20-30) лӮҙм—җм„ң кұ°мқҳ мқјм •##; SFTлҠ” progressively expand (45вҶ’75); RLVRмқҖ contract (70вҶ’15)
  • Figure 4(b) Frobenius norm $\|\Delta W_t\|_F$: ##OPDлҠ” RLVRліҙлӢӨ substantially larger## update accumulateн•ҳл©ҙм„ңлҸ„ stable rankлҠ” л№„мҠ· вҶ’ "small-update artifact" к°Җм„Ө кё°к°Ғ. OPDк°Җ м§„м§ңлЎң ##non-trivial updateмқҳ bounded spectral profile##мқ„ ліҙмһ„
  • Figure 4(c) Hill alpha (spectral tail estimator): OPD evolve mildly, SFT increases sharply, RLVR decreases over training -- auxiliary нҷ•мқё

мқҙлЎңмҚЁ OPDлҠ” SFTмҷҖ RLVRмқҳ ##temporal interpolation м•„лӢҳ##. н•ҷмҠө мҙҲкё°м—җ low-dimensional regimeмңјлЎң м§„мһ…н•ҳм—¬ ##к·ёкіім—җ лЁёл¬ёлӢӨ##.

4.2 Stability and Functional Sufficiency

Early subspace emergence (Figure 5)

Low dimensionality aloneмқҙ lockмқ„ мқҳлҜён•ҳм§„ м•ҠлҠ”лӢӨ (лӢӨлҘё low-rank subspaceлҘј rotationн•  мҲҳлҸ„). л”°лқјм„ң ##final low-dimensional update channel emerge time##мқ„ test:

$$\text{Sim}_K(t, t_{\text{end}}) = \frac{1}{K}\|V_K(t)^\top V_K(t_{\text{end}})\|_F^2 \tag{9}$$

$V_K(t)$лҠ” $\Delta W_t$мқҳ top-$K$ right singular subspace. 1м—җ к°Җк№Ңмҡ°л©ҙ updateк°Җ final update channelмқҳ singular directionмқ„ мқҙлҜё мӮ¬мҡ© мӨ‘.

кҙҖм°°: ##OPDлҠ” мІ« measured checkpointл¶Җн„° final $V_{16}$ subspaceмҷҖ align##; SFT, RLVRмқҖ gradually мҲҳл ҙ. ##OPDмқҳ low-dimensional channelмқҖ late assembleмқҙ м•„лӢҲлқј early emergence##.

Functional sufficiency (Figure 6)

мқҙ channelмқҙ descriptive onlyмқёк°Җ, sufficient for learningмқёк°Җ?

мӢӨн—ҳ мқҳлҸ„: ##K=16## (early stable rank scale, slightly below effective dimension)мңјлЎң ##stringent bottleneck## л¶Җм—¬. н•ҷмҠө ~20% мӢңм җм—җм„ң $V_{16}$ м¶”м¶ң вҶ’ мқҙнӣ„ н•ҷмҠө мӢң л§Ө gradient $g$лҘј ##$g \leftarrow g V_{16} V_{16}^\top$##лЎң project (Eq.10) вҶ’ мқҙ rank-16 constraint н•ҳм—җм„ң OPD vs SFT н•ҷмҠө.

кІ°кіј:

  • ##OPDлҠ” rank-16 bottleneck н•ҳм—җм„ң essentially unchanged## (AIME2024 avg@8 ~plainкіј лҸҷмқј)
  • ##SFTлҠ” л§Өм№ӯ windowм—җм„ң substantial degradation##
  • к°ҷмқҖ нҢЁн„ҙмқҙ м¶”к°Җ reasoning benchmark (AMC2023, AIME2025, Minerva, Beyond AIME)м—җм„ңлҸ„ hold (Appendix F, Figure 8)

мқҙ кІ°кіјк°Җ л’·л°ӣм№Ён•ҳлҠ” мЈјмһҘ: ##Locked update channelмқҖ OPDмқҳ н•ҷмҠөм—җ functionally sufficient##; к°ҷмқҖ constraintк°Җ SFTлҘј л§қк°ҖлңЁлҰјмңјлЎңмҚЁ sufficiencyк°Җ ##projection dimensionмқҳ generic propertyк°Җ м•„лӢҲлқј OPD-specific##мһ„мқ„ confirm. ##лҜёлһҳ OPD кө¬нҳ„мқҖ low-dimensionalityлҘј post-hoc spectral statisticмқҙ м•„лӢҢ monitored & exploited channelлЎң лӢӨлӨ„м•ј н•ңлӢӨ.##

5.1 Runtime Perturbations Preserve the Lock

(a) Token sparsification (Figure 7a)

Sampleл§ҲлӢӨ binary mask $m_t \in \{0, 1\}$. Top-KL retention: tokenмқ„ sampled-token student-teacher log-probability gap $r_t = \log\pi_\theta(a_t|s_t) - \log\pi_T(a_t|s_t)$лЎң ranking, top $\rho$ fraction мң м§Җ -- studentк°Җ teacher лҢҖл№„ over-confidentмқё token м„ нғқ. Random retention: $\rho \in \{0.25, 0.5\}$ uniform.

Compensation: $\tilde{r}_t = T m_t r_t / \sum_\tau m_\tau$.

кІ°кіј: лӘЁл“  sparsified variantк°Җ OPD baselineмқҳ stable-rank trajectoryлҘј ##closely track##. ##25% retention beyondлҠ” update scaleмқ„ лҚ” ліҖнҷ”мӢңнӮӨм§Җл§Ң spectral shapeлҠ” ліҖнҷ” вҮ’ low-rank profileмқҖ small set of high-KL tokenм—җ localizeлҗҳм§Җ м•ҠмқҢ##.

(b) Off-policy rollout (Figure 7b)

$y \sim \pi_T(\cdot | x)$ (teacher generates rollouts), studentк°Җ training forward passм—җм„ң teacher-generated tokenмқҳ log-probability кі„мӮ°. ##Training objectiveлҠ” pure OPD KL distillation мң м§Җ##. Importance-ratio clipping $\epsilon = 1.0$ (clipping inactive). Qwen3-32B teacher rollout, sglang router + 2 teacher replica.

кІ°кіј: Off-policy OPDк°Җ ##modestly larger update norm##мқ„ produceн•ҳм§Җл§Ң stable-rank trajectoryлҠ” ##nearly unchanged##. ##Rollout-policy ліҖнҷ”м—җ robust##.

5.2 Objective Mixing Changes Rank Dynamics (Figure 7c)

Advantage level interpolation:

$$A_i^{(\alpha)} = \alpha A_{i, \text{OPD}} + (1-\alpha) A_{i, \text{RLVR}} \tag{11}$$

$A_{i, \text{OPD}} = -r_i$ (sampled-token teacher-correction), $A_{i, \text{RLVR}}$ = GRPO advantage from math accuracy reward broadcast to response tokens. $\alpha \in \{0.75, 0.50, 0.25, 0.05, 0.01\}$.

кІ°кіј:

  • OPD-dominant mixture (Оұ нҒ° к°’) вҶ’ OPD-like stable-rank trajectory мң м§Җ
  • ##Оұ = 0.05/0.01 (OPD weak)м—җм„ң trajectoryк°Җ deviation##, лӢӨлҘё spectral regimeмңјлЎң м§„мһ…
  • Token sparsificationмқҙлӮҳ off-policy generationкіј лӢ¬лҰ¬, ##gradient source мһҗмІҙлҘј ліҖкІҪ##

Mechanistic view: OPD gradient = $g_{\text{OPD}} = \sum_t J_t^\top \delta_t$, $\tilde{g}_{\text{OPD}} = \sum_t m_t J_t^\top \delta_t$. Token gradientк°Җ dominant update subspace кіөмң н•ҳл©ҙ ##maskingмқҖ second momentлҘј primarily rescale##:

$$\mathbb{E}[\tilde{g} \tilde{g}^\top] \approx c \mathbb{E}[g_{\text{OPD}} g_{\text{OPD}}^\top] + \text{noise} \tag{13}$$

Off-policy rolloutмқҖ sampling distributionмқ„ ліҖкІҪн•ҳм§Җл§Ң teacher-token gradient source лҸҷмқј. Objective mixingмқҖ source мһҗмІҙ ліҖкІҪ:

$$g_\alpha = \alpha g_{\text{OPD}} + (1-\alpha) g_{\text{RLVR}}, \quad \Sigma_\alpha \approx \alpha^2 \Sigma_{\text{OPD}} + (1-\alpha)^2 \Sigma_{\text{RLVR}} + \alpha(1-\alpha) \Sigma_{\text{cross}} \tag{14}$$

OPD termмқҙ weakн•ҙм§Җл©ҙ ##dominant covariance geometry ліҖкІҪ##. ##LockмқҖ objective-sensitive, not runtime-induced.##

Implication: ##нҡЁкіјм Ғмқё OPD recipeлҠ” token coverageлӮҳ rollout generationмқҙ м•„лӢҲлқј objective-induced update geometryлҘј regulate##н•ҙм•ј н•ңлӢӨ.

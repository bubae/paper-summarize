하나의 도구로 자동 최적화하다

왜 필요한가?

AI 모델의 성능은 모델 구조만큼이나 학습 데이터의 선택과 배합에 크게 좌우됩니다. 그런데 데이터를 "잘" 고르고 배합하는 방법이 10가지 넘게 있고, 모두 서로 다른 코드로 개발되어 있어서 비교도, 재현도, 실용적 적용도 어렵습니다.


DataFlex는 이 문제를 해결하기 위해, 가장 인기 있는 LLM 훈련 도구(LLaMA-Factory) 위에 모든 데이터 최적화 방법을 통합한 프레임워크를 만들었습니다.

세 가지 데이터 최적화 방법을 하나로

Selection
좋은 데이터만 골라서
학습 (7가지 알고리즘)
Mixture
데이터 종류별 비율
자동 조절 (2가지)
Reweight
각 데이터의 중요도
실시간 조절 (1가지)

핵심 결론: 데이터를 "정적으로" 그냥 쓰는 것보다, DataFlex에서 제공하는 "동적" 방법을 쓰면 일관되게 더 좋은 성능을 얻을 수 있습니다. 기존 LLaMA-Factory 설정에 YAML 몇 줄만 추가하면 됩니다.

한계점

DataFlex 자체는 새로운 데이터 최적화 "알고리즘"을 만든 것이 아니라, 기존 알고리즘들을 하나의 "도구 상자"에 모은 것입니다. 또한 평가가 MMLU 등 소수 벤치마크에 국한되어 있고, LLaMA-Factory에 의존하는 한계가 있습니다.

DataFlex = 10가지 방법 통합 + LLaMA-Factory 호환 + 오픈소스
